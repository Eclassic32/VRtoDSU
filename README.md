# VRtoDSU

This app is ment to turn your VR controllers into virtual Joycons with Motion and Rumble

## How to use
-# FIX: Some info here

1. Download and set up Host app from releases
2. Launch Host app and get addresses
3. Open web page with Meta Quest Browser, Chrome or other supported browser (apparently Firefox doesn't work?)
4. Go to controller settings in your Yuzu/Ruyjinx/forks
5. Set the controller as "whatever should be here" and emulation to "JoyCon Pair"
6. Enable Motion and Rumble, Go to Motion settings
7. Enable Cemuhook compatible motion
8. Enter DSU address from Host app, `1` into `Controller Slot` and `2` into `Right Slot`
9. Apply, save and use

> To stop using the app, just close its window.
